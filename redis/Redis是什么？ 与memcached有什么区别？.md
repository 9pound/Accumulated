# 什么是Redis？与memcached有什么区别？

## Redis

**简介：**Redis是一个速度非常快的非关系（nosql）内存数据库，它可以存储键（key）与5种不同类型的值之间的映射（mapping），同时可以将存储在内存中的键值对数据持久化到硬盘。可以使用复制特性来扩展读性能，还可以使用客户端分片来扩展读写性能。

**特点：**

1. 支持5中不同类型的数据结构

2. 拥有两种不同形式的持久化方案

   - 时间点转储：
   - 将所有修改了数据库的命令都写入一个只追加文件里面，根据数据的总要程度可以设置为“从不同步”、“每秒同步一次”、“每写入一个命令就同步一次”

3. 主从复制



### **redis与memcached的区别**

- 性能相差无几

- redis可以用作主数据库使用，也可以作为其他存储系统的辅助数据库使用

- memcached只能存储普通的字符串键。Redis除了能存储普通的字符串键之外，还可以存储其他4种数据结构

- 删除元素的时候，memcached采用黑名单来隐藏列表里面的元素，而redis的List和set允许用户直接添加或者删除元素

- Redis的聚合数据可以是整数或浮点数，而memcached的聚合数据只能是整数
