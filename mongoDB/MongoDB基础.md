# MongoDB

## 是什么

MongoDB是一个基于分布式文件存储的数据库。由C++语言编写。旨在为WEB应用提供可扩展的高性能数据存储解决方案。

MongoDB是一个介于关系数据库和非关系数据库(nosql)之间的产品，是非关系数据库当中功能最丰富，最像关系数据库的。

MongoDB是一个面向集合的，模式自由的文档型数据库。



特点：

不许要固定的表结构，通常也不存在连接操作。
