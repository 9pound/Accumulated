2. 对象的访问定位的两种方式？

   **句柄:**在Java堆中划分一块内存作为句柄池，栈上的reference数据存储对象的句柄地址，句柄包含到对象实例数据和类型数据的指针。

   对象被移动时，只会改变句柄中实例数据指针，而reference本生不需要修改。

   **直接指针:**直接在reference中存储对象的地址（对象实例数据的地址）

   速度更快，节省了一次指针定位的时间
