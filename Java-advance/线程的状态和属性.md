# **线程的状态和属性**



## 状态

**线程可以有如下6种状态**
	新创建：New
	可运行：	Runnable
	被阻塞：	Blocked
	等待： 	Waiting
	计时等待	：Timed waiting
	被终止：	Terminated
	要确定一个线程的当前状态，可调用getState方法

## 线程属性

**4.1线程优先级**
	（1）在java程序设计语言中，每一个线程有一个优先级。默认情况下一个线程继承它的父线程的优先级
	（2）可以用setPriority方法提高或降低任何一个线程的优先级
**4.2守护线程**
	（1）可以通过调用setDaemon（true）将线程转换为守护线程，
	（2）守护线程的作用：唯一作用是为其他线程提供服务。
	（3）当只剩下守护线程时，虚拟机就退出了
	（4）守护线程应该永远不去访问固有资源，因为它会在任何时候甚至在一个操作的中间发生中断。
