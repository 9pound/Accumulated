# 进程、线程、协程、多任务、多进程、多线程

**进程**

​	（1）本质：进程是操作系统对一个正在运行的程序的抽象

**线程**

​	（1）本质：线程是进程中的一个控制流

​	（2）补充：一个进程实际上是由多个称为线程的执行单元组成，可以说线程是轻量级的进程



**协程：**

**多任务**：在同一时刻运行多个程序的能力。

**多线程**：一个程序同时执行多个任务。

**多线程程序**：可以同时运行一个以上线程的程序。

**多进程与多线程的区别**：

- 本质区别在于每个进程拥有自己的一整套变量，而线程则共享数据（当前进程范围内，内存句柄，文件句柄等）。
- 每个线程都有各自的程序计数器，栈以及局部变量等

**Java程序运行原理**：Java命令会启动java虚拟机，启动JVM，等于启动了一个应用程序，也就是启动了一个进程。该进程会自动启动一个 “主线程” ，然后主线程去调用某个类的 main 方法。



## 什么是线程

1、使用线程给其他任务提供机会
 	如果需要执行一个比较耗时的任务，应该使用独立的线程
2、在一个单独的线程中执行一个任务的简单过程
	1）将代码移动到实现了Runnable接口的类的run方法中，Runnable接口非常简单，只有一个run（）方法
	2）创建一个类对象
	3）由Runnable创建一个Thread对象
	4）启动线程
3、当发生InterrupterException异常时，run方法将结束执行
4、也可以通过构建一个Thread类的子类定义一个线程，不过这个方法已不再推荐
5、如果有多个任务，要为每一个任务创建一个独立的线程，可以使用线程池来解决这个问题
