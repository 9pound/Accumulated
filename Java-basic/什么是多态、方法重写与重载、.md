### 什么是多态、方法重写与重载、





**方法覆盖**

（1）含义：父子类中具有相同方法签名的方法。

（2）本质：是子类重新定义了父类的方法

**补充**：只有非private方法才可以被覆盖（在导出类中，对于基类中的private方法，最好采用不同的名字）

**方法重载**

（1）含义：同一个类中，如果多个方法具有相同的名字、不同的参数，变产生了重载。

（2）本质：用相同的词表达多种不同的含义

**补充**：

（1）java允许重载任何方法。

（2）方法签名：即方法的方法名及参数列表。返回类型不是方法签名的一部分。

（3）重载解析



### 如何区分重载方法？

（1）参数类型 

（2）参数的数量

（3）参数的顺序



**注意**：方法的返回值不能用来区分重载方法
