**8种基本数据类型**

（4种整型，2种浮点类型，1种字符类型（表示Unicode编码），一种boolean类型（表示真值））

|  类型   | 长度(字节) |     范围      |
| :-----: | :--------: | :-----------: |
|  byte   |     1      |   -128~127    |
|  short  |     2      | -32768~32767  |
|   int   |     4      |   正负21亿    |
|  long   |     8      |               |
|  float  |     4      | 有效位数为6~7 |
| double  |     8      | 有效位数为15  |
| boolean |     1      |               |
|  char   |     2      |               |

注意：

- long有后缀L或l，float类型有后缀F或f（没有后缀的浮点数默认为Double，），也可以在浮点数值后加后缀D或者d表示double类型。
- 十六进制前有前缀0x或0X，八进制有前缀0，二进制有前缀0b或0B（建议不要使用八进制容易混淆）
- 可以为数字字面量加下划线 1_000_000 或者 0b1111_0010_0001
- 为什么double被称为双精度数因为double表示数值精度是float类型的两倍
